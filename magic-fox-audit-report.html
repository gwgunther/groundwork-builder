Content Gaps

Seven opportunities the current site leaves on the table.

High Priority

Before/After Gallery

The #1 conversion driver for orthodontic practices. Yelp has 37 photos proving results exist — they need to be on the website with MedicalProcedure schema.

Medium Priority

Insurance & Financing Page

"Does insurance cover braces" is a top search query. A dedicated page with accepted plans, zero-interest financing details, and FAQ schema would capture this traffic.

Medium Priority

FAQ Sections on Service Pages

FAQPage schema drives featured snippets. Each service hub (Invisalign, adult ortho, early interceptive) should have 5-8 FAQs with structured data.

Medium Priority

Spanish & Vietnamese Landing Pages

Trilingual care is a top differentiator, but zero multilingual content exists. Even one page per language would capture underserved search intent.

Medium Priority

Online Scheduling Integration

Reduce friction from discovery to booking. The site uses Flatpickr for date picking — a proper scheduling widget would convert more visitors.

Lower Priority

Team/Staff Page

Patients mention the welcoming staff in reviews. Showcasing the full team beyond the doctors builds connection and trust.

07

SEO Opportunities

The 92 SEO score is a launchpad, not a ceiling.

1. Dominate the local pack for "orthodontist huntington beach"

418 reviews at 5.0 stars + comprehensive LocalBusiness schema = perfectly positioned. Ensure NAP consistency across all directories, add breadcrumb schema to inner pages, and build location-specific content for surrounding neighborhoods.

2. Build out the Invisalign page to 1,500+ words

The current page exists but needs depth — comprehensive treatment info, FAQ schema, before/after photos, and strong CTAs to compete for "invisalign huntington beach" (high-value keyword, ~$15+ CPC).

3. Add AggregateRating schema to every service page

The 5.0/418 rating currently lives only on the org schema. Extending it to service pages via review snippets could earn star ratings in SERPs for service-specific queries.

4. Expand geo-targeted content beyond Fountain Valley

The /fountain-valley-orthodontist page is smart. Replicate for Westminster, Garden Grove, Midway City, and Costa Mesa — each with unique local content and schema areaServed markup.

5. Capture multilingual search intent

"Ortodoncista huntington beach" and Vietnamese-language orthodontic queries have minimal competition. Even basic pages in each language could own these niches.

08

Redesign Recommendations

What the data says to prioritize.

1. Migrate from Webflow to Astro static build

This single change addresses the #1 problem. Eliminating Webflow runtime, reducing JS from 6+ scripts to essentials, and optimizing images should push Performance from 42 → 90+. Deploy on Cloudflare Pages for global edge delivery.

2. Replicate the existing schema — then extend it

The current JSON-LD is excellent. Carry it forward exactly, then add: FAQPage schema on service pages, BreadcrumbList on all inner pages, MedicalProcedure schema on before/after gallery, individual doctor image properties.

3. Build a before/after gallery as the top content addition

37 photos exist on Yelp. Curate the best into a dedicated gallery with treatment type filters. This is the highest-impact content gap for orthodontic conversion.

4. Amplify the 5.0 / 418 review story

This rating is the practice's single strongest asset. Feature it in the hero, on every service page, and in schema. The current site underplays it — the redesign should make it unmissable.

5. Preserve all tracking and verification

GTM (GTM-TG4XBTBN), Facebook Pixel (1911570572950976), Google Call Tracking, and site verification (thRNL5qDaI7D_p8YeQRXeq6unv1vQM5r71SIlKHw4z8) must migrate cleanly. 301 redirects for all 25 existing URLs.
